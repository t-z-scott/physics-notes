# About
This website was created off my Physics 2 notes from my Obsidian repository. Physics 2 was difficult for me, so I decided to post these in a way that people can access easily.

This website was created with [MkDocs](https://www.mkdocs.org) and imports [mdx_math](https://github.com/mitya57/python-markdown-math) to render the math equations (Thank you to Dmitry Shachnev). The textbook for this class was "Cutnell and Johnson Physics, 12th edition" by Young and Stadler. Credit for the slides goes to Dr. Nadra Sakr.
